# transferApiTests
Test suites for Transfer api
### Requirements:
  * npm
  * Node.js (v13.9.0)

[How to install npm + node](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
## How to install
Install needed packages via `npm i` in root folder.
## How to run tests:
Run test via `npm run test`
